# 비용 안내 — Subtitle Marketing Studio

자막 1개 → 15개 산출물 한 번 돌릴 때 드는 LLM 비용입니다.

## 핵심: 카드뉴스도 API 비용 0원

카드뉴스 5종(Threads x2, Instagram x2, KakaoTalk)은 **Jinja2 + Playwright 로컬 캡처** 방식입니다.
- LLM이 만든 JSON → Jinja2 HTML 템플릿 → Playwright headless Chromium → PNG
- **추가 API 호출 없음**. Chromium은 로컬에서 실행.
- 비용은 LLM의 JSON 생성 텍스트 호출 1회뿐.

이미지 LLM(Nano Banana / gpt-image-2 등)은 사용하지 않습니다.

## 15개 전체 비용 (DeepSeek v4 flash)

`deepseek-v4-flash` 모델, **호출 1건당 약 $0.0005 ~ $0.003** (출력 토큰 길이에 따라).

| 그룹 | 호출 수 | 평균 출력 토큰 | USD | KRW |
|---|---:|---:|---:|---:|
| 블로그 9종 | 9 | ~3,000 토큰 | ~$0.018 | **~27원** |
| 보도자료 1종 | 1 | ~2,000 토큰 | ~$0.002 | **~3원** |
| 카드뉴스 5종 (JSON) | 5 | ~5,000 토큰 | ~$0.020 | **~30원** |
| **15개 전체** | **15** | — | **~$0.04** | **~60원** |

자막 1개 전체 처리 ≈ **60원 미만** (편의점 사탕 1개 가격).

## 환율 설정

`USD_KRW` 상수는 [../core/user_settings.py](../core/user_settings.py)에 정의돼 있습니다. 환율 변동 시 직접 수정:

```python
USD_KRW = 1504.71  # 2026-05-27 기준
```

## 부분 실행 시

마음에 안 드는 산출물만 재실행:

| 작업 | 비용 |
|---|---:|
| 블로그 1편 재실행 | ~3원 |
| 보도자료 재실행 | ~3원 |
| 카드뉴스 1세트 재실행 (LLM JSON + Playwright) | ~6원 |

## 비용 비교

- **현재 (Playwright 로컬 렌더링)**: 자막 1개 ≈ 60원
- 만약 이미지 LLM 사용 시 (Nano Banana, $0.04/장): 자막 1개 ≈ 1,300원 + 한글 깨짐 위험

차이가 큰 이유는 **카드 디자인을 LLM이 그리는 게 아니라 사전에 설계된 Jinja2 템플릿에 끼워 넣기** 때문입니다.

## 정확한 청구 비용 확인

위 표는 추정치입니다. 실제 청구는 Ubion LiteLLM 대시보드(`http://192.168.50.119:4000/ui/`)에서 API 키별 사용량 조회로 확인하세요.

## 이미지 LLM을 다시 쓰고 싶다면

[../core/image_render.py](../core/image_render.py) 모듈이 코드에 남아 있어 opt-in으로 재활성화 가능합니다. 단, 한글 렌더링 한계와 카드 디자인 일관성 부족 때문에 권장하지 않습니다.
