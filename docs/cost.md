# 비용 안내 — Subtitle Marketing Studio

자막 1개 → 15개 산출물 한 번 돌릴 때 드는 LLM 비용입니다. 환율은 [core/user_settings.py](../core/user_settings.py)의 `USD_KRW` 상수에서 관리(2026-05-27 기준 **1 USD = 1,504.71 KRW**, 갱신은 수동).

| **카드뉴스 합계** | **22장** | **$0.88** | **~1,324원** |

## 텍스트 9종 + 보도자료 — DeepSeek v4-flash

`deepseek-v4-flash` 모델, **호출 1건당 약 $0.0005 ~ $0.002** (출력 토큰 길이에 따라). 15종 중 텍스트 산출물(블로그 9개 + 보도자료 1개 = 10건) 합계 **<$0.02 (~30원 미만)**.

## 자막 1개 전체 처리

| 항목 | USD | KRW |
|---|---:|---:|
| 카드뉴스 이미지 22장 | $0.88 | ~1,324원 |
| 텍스트 산출물 10건 | <$0.02 | <30원 |
| **총합** | **~$0.90** | **~1,350원** |

라면 1봉지 가격으로 자막 1개를 15개 채널-레디 산출물로 변환합니다.

## 비용 표시 위치

- **좌측 스튜디오 패널 상단**: `💰 예상 이미지 비용 ~$0.88 (≈1,324원) · 환율 1,504.71원/USD`
- **실행 후 같은 자리**: 실제 누적 사용액으로 자동 갱신
- **각 카드뉴스 스튜디오 산출물 패널**: 카드 미리보기 위에 `🖼 Nano Banana 이미지 생성 · 카드 5장 · 비용 $0.200 (≈301원)` 라인

## 환율 갱신 방법

환율이 크게 변하면 [core/user_settings.py:14](../core/user_settings.py#L14)의 `USD_KRW` 값만 수정:

```python
USD_KRW = 1504.71  # 2026-05-27 → 새 환율로 교체
```

Streamlit이 자동 리로드합니다.

## 모델 변경 방법

이미지 모델: [core/user_settings.py:13](../core/user_settings.py#L13)의 `IMAGE_MODEL` 변경 (예: `gpt-image-2`, `gemini-3-pro-image-preview`).
모델별 단가는 [Ubion LiteLLM Migration Kit](../_context/litellm_kit/README.md) 참조.

## 비용 절감 팁

1. **카드 1장 재실행만**: 카드 카드 단위 재생성은 아직 미지원 — 스튜디오 전체 재실행만 가능. 한 카드의 텍스트가 맘에 안 들면 그 스튜디오 전체 재실행.
2. **`재실행` vs `전체 16개 실행`**: 카드뉴스 5개 중 1개만 다시 뽑을 거면 `재실행` 버튼이 4배 저렴.
3. **블로그만 필요한 경우**: 카드뉴스 5개를 한꺼번에 `🗑 초기화` 해두면 이미지 비용 0원으로 텍스트만 생성 가능 (앞으로 부분 실행 옵션 추가 예정).
4. **환율 헤지 안 됨**: 결제는 USD로 사내 LiteLLM 프록시 정산이라 환율 변동 위험은 있음.
