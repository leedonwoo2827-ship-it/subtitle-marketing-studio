```
{
  "cards": [
    {
      "role": "헤드라인",
      "icon": "💬",

      "tagline": "한 줄 여운 (선택)"
    }
  ],
  "caption": "카카오 메시지 본문 텍스트 (200자 이내) — 이미지엔 안 박힘, 카드와 함께 전송",
  "button": "최종 CTA 버튼 라벨 (8자 이내)"
}
```

규칙:
- `cards` 1~3개. `body` 모든 카드 필수, 2~3문장 또는 개조식.
- 1장은 항상 후킹·헤드라인. 마지막 카드에만 `button` 필드.
- 각 카드 headline에 이모지 1~2개.
- 톤: 친구에게 보내는 메시지, 짧고 즉각적.
- 자막 원문 그대로 인용 금지. 해시태그 사용 안 함.
