스키마:
```
{
  "cards": [
    {
      "icon": "이모지 1개 — 상단 큰 일러스트",
      "headline": "한 줄 헤드라인 (이모지 없이, 20자 이내)",
      "blocks": [
        { "subhead": "소제목 1", "body": "2~3문장. 마침표 뒤 자동 줄바꿈." },
        { "subhead": "소제목 2", "body": "2~3문장. (선택)" }
      ],
      "tagline": "한 줄 여운 (선택)"
    }
  ],
  "caption": "카카오 메시지 본문 (200자 이내) — 이미지엔 안 박힘",
  "button": "버튼 라벨 (8자 이내)"
}
```

규칙:
- `cards` 1~3개.
- 각 카드 `blocks` 는 **1~2개** (카드 사이즈가 작으니 짧게).
- `headline` 에 이모지 넣지 말 것.
- 톤: 친구에게 보내는 메시지, 짧고 즉각적.
- 자막 원문 그대로 인용 금지. 해시태그 사용 안 함.
