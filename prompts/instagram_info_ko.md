      "body": "3~5문장 또는 개조식 4~5줄. 카드의 핵심 메시지를 풍부하게 풀어쓰는 본문.",
      "tags": ["짧은태그1", "짧은태그2"],
      "tagline": "마지막 한 줄 강조 (선택)"
    },
    {
      "role": "본론 1",
      "icon": "이모지 1개", "headline": "📌 ...", "subhead": "...", "body": "...",
      "tags": [...], "tagline": "..."
    },

      "icon": "이모지 1개", "headline": "📍 ...", "subhead": "...", "body": "...",
      "tags": [...], "tagline": "..."
    },
    {

- 각 카드 `headline` 시작에 이모지 1~2개.
- `tags` 는 2~3개 짧은 키워드(#없이).
- 톤: 친근·존댓말. 자막 원문 그대로 인용 금지.
