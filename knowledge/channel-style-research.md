# Channel Style Research — Subtitle Marketing Studio

자막(Raw Data)을 **채널별로 그대로 업로드 가능한 산출물**로 재가공할 때 각 채널의 톤·길이·포맷·시각 규칙을 정의한 살아있는 가이드. 15개 스튜디오가 각자 `channel_section` 키로 해당 섹션만 발췌해 프롬프트에 주입한다.

---

## PART 0. 사용 철학

- **원칙 1**: 자막 원문은 외부에 노출하지 않는 Raw Data. 산출물은 채널에 그대로 붙여넣기·업로드 가능한 형태.
- **원칙 2**: 채널이 다르면 톤·길이·구조가 다르다. 같은 자막이 15번 다르게 가공된다.
- **원칙 3**: 카드뉴스(Threads, Instagram, KakaoTalk)는 **카드 안에 모든 텍스트가 들어가야** 한다. 캡션도 마지막 카드 안에.
- **원칙 4**: 블로그 9종은 플랫폼별 톤·길이가 다르다. 마케터는 자막 1개로 15종 산출 후 골라 쓴다.
- **업데이트 마커**: 🔎 조사 필요 / ✅ 검증됨 / ⚠ 변동성 높음

## PART 2. 카드뉴스 채널 상세 스펙

### Threads 간결형 카드뉴스
- 📐 1080×1350 [4:5] **3장**, 카드당 텍스트 ≤ 80자
- 🎭 톤: 트위터/Threads 후속, 짧고 직설적·반말 가능, 1인칭
- 🪝 1장: 후킹 한 줄(질문·숫자·반전) / 2장: 핵심 1개 / 3장: CTA·코멘트 유도
- 💬 카드 안 CTA: "댓글로 의견 들려주세요" / "리포스트 환영"
- #️⃣ 마지막 카드 하단 1~3개 (#스레드 #쓰레드 등)
- 👥 20~35세, 빠른 소비, 토론·반응형
- 📆 발행: 평일 12~14시, 19~21시
- 🔎 Threads 알고리즘은 답글·리포스트 가중치 높음 (2026 기준)

### Threads 인사이트형 카드뉴스
- 📐 1080×1350 [4:5] **5장**, 카드당 텍스트 ≤ 100자
- 🎭 톤: 진중·정보성, 데이터 1~2개 포함
- 🪝 1장: 후킹·문제제기 / 2~4장: 인사이트 단계 전개 / 5장: 결론·CTA
- 💬 카드 안 CTA: "이 글에서 가장 와닿은 한 줄을 댓글로"
- #️⃣ 5장 하단 2~3개
- 👥 25~40세, 직장인·전문가, 깊이 있는 콘텐츠 선호
- 📆 발행: 평일 오전 8~9시, 오후 6~7시

### Instagram 정보형 카드뉴스
- 📐 1080×1350 [4:5] **5장**, 카드당 텍스트 ≤ 120자
- 🎭 톤: 친근·존댓말, 데이터·정리 중심
- 🪝 1장: 표지 후킹(숫자·반전·호기심) / 2~4장: 본론 3단계 / 5장: 요약+CTA+캡션 내장
- 💬 **5장 카드 안에 캡션 본문(150자)·해시태그 10~15개·CTA 모두 포함**
- #️⃣ 5장 하단에 한글·영문 혼합 10~15개
- 👥 25~40세, 정보 탐색·저장형
- 📆 발행: 평일 19~21시, 주말 11~13시
- ⚠ 캐러셀 도달이 단일 이미지 대비 ↑ (2026 기준)

### Instagram 스토리텔링형 카드뉴스
- 📐 1080×1350 [4:5] **7장**, 카드당 텍스트 ≤ 100자
- 🎭 톤: 1인칭·서사·감성, 빈티지·포토에세이 분위기
- 🪝 1장: 도입 한 문장 / 2~6장: 시간 흐름·에피소드 / 7장: 결말·여운+캡션 내장
- 💬 7장 카드 안에 캡션 본문(200자)·해시태그 5~10개 포함
- #️⃣ 5~10개, 감성·문학 키워드 우선
- 👥 25~45세, 에세이·문학 선호층
- 📆 발행: 평일 21~22시, 일요일 오전 10시

### 카카오톡 카드뉴스 (친구톡 / 채널톡)
- 📐 800×800 [1:1] **1~3장**
- 🎭 톤: 친구에게 보내는 메시지, 짧고 즉각적
- 🪝 1장: 한 줄 헤드라인 / 2~3장: 핵심 1개씩
- 💬 카드 안 CTA: 버튼 라벨 가정 ("자세히 보기" / "지금 확인")
- #️⃣ 사용 안 함
- 👥 친구·구독자 직접 발송
- ⚠ 카카오 비즈니스 채널은 사전 템플릿 승인 필요. 본 산출물은 가이드용·시안용.

### 보도자료
- 📐 DOCX, 800~1,500자, 5단 구성
- 🎭 톤: 3인칭 객관, 형용사·과장 금지, 수치·인용 중심
- 구조: 헤드라인 / 부제 / 리드(누가·무엇·언제·어디·왜) / 본문(배경·핵심·인용·기대) / 회사 소개 / 문의·발신
- #️⃣ CTA·해시태그 사용 안 함
- 👥 기자·에디터
- 📆 발행: 화·수·목 오전 10~11시

## PART 3. 블로그 9종 상세 스펙

### 블로그 네이버 정보형
- 📐 2,500자, H2 3~5개·H3 보조
- 🎭 톤: 존댓말, 정보 전달, 첫 200자에 결론 노출(SEO 스니펫)
- 🪝 후킹: 숫자형(`5가지 이유`) > 질문형 > 단언형
- 💬 CTA: 댓글·구독, 본문 중간 1회·말미 1회
- #️⃣ 본문 하단 5~10개, 한글·영문 혼합
- 👥 25~45세, 정보 탐색·문제 해결
- ⚠ 네이버는 마크다운 미지원 — HTML 또는 리치텍스트 복붙
- 🔎 2026 네이버 검색은 AI 인용 친화 포맷(Q&A·표) 가중치 ↑

### 블로그 네이버 일상형
- 📐 1,500자, H2 2~3개, 짧은 단락(2~3문장)
- 🎭 톤: 1인칭(저는·우리는), 일기·체험 톤, 사진 캡션 같은 자연스러움
- 🪝 후킹: 일상 장면 묘사·감정·소소한 발견
- 💬 CTA: "여러분은 어떠세요?" 질문 1개
- #️⃣ 본문 하단 5~7개
- 👥 30~50세, 일상·라이프스타일 탐색

### 블로그 네이버 리뷰형
- 📐 2,000자, H2 4~5개 (소개·장점·단점·총평·추천 대상)
- 🎭 톤: 존댓말, 실사용 후기, 균형 잡힌 평가
- 🪝 후킹: "솔직한 후기" / "한 달 써봤습니다" / 별점
- 💬 CTA: "다음 리뷰가 궁금하다면 구독"
- #️⃣ 본문 하단 7~10개 (제품명·카테고리)
- 👥 30~45세, 구매 의사결정 단계

### 블로그 티스토리 SEO형
- 📐 3,500자, H2 5~7개·H3 풍부, 표·리스트 적극 활용
- 🎭 톤: 정보성, 키워드 밀도 ≥ 2%, 메타 디스크립션 명시 권장
- 🪝 후킹: 검색 의도 직답("X란 무엇인가" / "X 하는 법")
- 💬 CTA: 관련 글 링크 다수
- #️⃣ 본문 끝 10~15개
- 👥 검색 유입 의도, 25~45세
- 🔎 구글·다음 검색 노출 우선

### 블로그 Medium 에세이형
- 📐 2,500자, 짧은 단락(2~4문장), H2 3~4개
- 🎭 톤: 인사이트·통찰, 격식 있는 1인칭, 도입 강력
- 🪝 후킹: 역설·반전·개인 경험 후 일반화
- 💬 CTA: 클랩·하이라이트 유도 ("Highlight what resonates")
- #️⃣ Medium 태그 시스템 5개 이내
- 👥 25~45세, 글로벌 지식 노동자
- ⚠ 한글 Medium도 가능하지만 영문 비중 높음

### 블로그 브런치 감성형
- 📐 2,500자, 호흡 긴 단락(3~6문장), H2 2~4개
- 🎭 톤: 문학적, 사색·감성, 비유·은유 활용
- 🪝 후킹: 풍경·감정·기억 묘사로 도입
- 💬 CTA: 직접 CTA 없음. 다음 글 예고로 마무리
- #️⃣ 브런치 키워드 3~5개
- 👥 30~50세, 에세이·문학 독자

### 블로그 개인 기술블로그 회고형
- 📐 1,500자, 짧은 단락, H2 2~3개
- 🎭 톤: 회고·다이어리, 짧고 정직, 자조적 유머 OK
- 🪝 후킹: 시간 표현("이번 주", "지난달") + 한 가지 배움
- 💬 CTA: 없음 (자기 기록 성격)
- #️⃣ 3~5개
- 👥 동료 개발자, 본인 미래

### 블로그 칼럼 시사형
- 📐 2,500자, H2 3~4개, 논거·반증 구조
- 🎭 톤: 시사 관점, 객관·날카로움, 3인칭 우세
- 🪝 후킹: 최근 사건·통계 인용으로 시작
- 💬 CTA: "당신의 견해는?" 토론 유도
- #️⃣ 시사 키워드 5~7개
- 👥 30~50세, 시사·정책 관심층

### 블로그 워드프레스 영문 SEO
- 📐 3,000자 (영문 기준), H1 1개·H2 5~7개·H3 풍부
- 🎭 톤: Informative, 2인칭(You), 짧은 문장
- 🪝 후킹: Question or stat in first sentence
- 💬 CTA: Subscribe / Comment / Share
- #️⃣ 영문 태그 5~10개
- 👥 글로벌 검색 유입
- 🔎 Google 검색 최적화 우선

## PART 4. 자막 1개 → 채널 발행 플레이북

자막 1개로 15개 산출 후 마케터가 1~3채널 선택:

- **B2C 일상 브랜드**: Instagram 스토리텔링형 + 블로그 네이버 일상형 + Threads 간결형
- **B2B 인사이트 콘텐츠**: LinkedIn 대체 = Threads 인사이트형 + 블로그 Medium 에세이형 + 블로그 티스토리 SEO형
- **신제품·이벤트 알림**: 카카오톡 카드뉴스 + Instagram 정보형 + 보도자료
- **기술 콘텐츠**: 블로그 개인 기술블로그 회고형 + Threads 인사이트형 + 블로그 티스토리 SEO형

**재사용 규칙**: 자막 1개 → 채널 최대 3개, 채널 간 발행 ≥ 7일 간격, 중복 톤 회피.
